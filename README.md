## 作ったもの

### 添加物スキャナー
カメラで食品表示ラベルを読み取ると、添加物情報を表示してくれるアプリを開発しています。アプリは iOS版がリリース済みで、Web、Android(TWA)でもリリース予定です。

[添加物スキャナー](https://foodadditive.app/)
